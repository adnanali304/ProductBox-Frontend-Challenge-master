Productbox Frontend Code Challenge
==================================

No third-party framework or library is used!

Cart
--------
- Add items 
- Remove Items
- Modify item quantity
- Real-time update when the cart is updated!

Items
--------
- List Items,
- Sort By (Name, Price)
- Add Items
- Real-time update  when a new Item is added!


js/events.js
--------
